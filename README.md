# Belajar CI 4 Poltek

## Install Library CI
1. Buka folder projek pada terminal (CMD)
2. Jalankan perintah `compoeser install`
3. Tunggu sampai install selesai

## Running CI 4
1. Buka folder projek pada terminal (CMD)
2. Jalankan perintah `php spark serve`
3. Buka browser dan tuliskan URL `localhost:8080`
